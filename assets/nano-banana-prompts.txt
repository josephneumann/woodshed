THE WOODSHED — ASSET GENERATION PROMPTS (Nano Banana / Gemini image)
=====================================================================

HOW TO USE
1. Do everything in ONE session so the style stays locked.
2. Generate prompt 01 (the master plate) first. Regenerate until the paper,
   ink and line weight feel right. Everything else copies it.
3. For every later prompt, ATTACH the master plate image as a reference and
   keep the "same style" opening line.
4. Aspect: 4:3 for plates, 1:1 for stamps, 16:5 for the hero (or widest offered).
   Highest resolution available. Save as PNG (JPG fine for the hero).
5. Save each file with the EXACT filename given, into the folder given.
   I'll wire them into the app from those paths.

If the model adds text or extra colors, reply: "remove all text and lettering;
monochrome sepia ink only, except one amber-brass accent."

NEGATIVE / AVOID (append or paste if the model supports it)
photograph, photorealistic, 3D render, gradients, glow, neon, cartoon, anime,
flat vector, thick uniform outlines, text, letters, numbers, watermark,
signature, frame border, busy background
